# PriceWise AI (MVP)

PriceWise AI is an MVP (Minimum Viable Product) of an Agentic RAG-based pricing strategy chatbot for management decision support.

The system combines:

- RAG (Retrieval-Augmented Generation)
- FAISS vector search
- Management frameworks
- Pricing strategy knowledge
- Gemini AI reasoning

## Features

- Pricing strategy recommendation
- Competitor and substitute analysis
- Premium vs competitive pricing support
- Strategic management framework integration
- Interactive chatbot workflow

## Frameworks Used

- Porter’s Five Forces
- VRIO / RBV
- Blue Ocean Strategy
- PESTEL
- FMEA
- Pricing Strategy Models

## Tech Stack

- Python
- Google Colab
- FAISS
- Sentence Transformers
- Gemini API
- Pandas

## Status

This project is currently an MVP developed for experimentation, learning, and future academic research.
